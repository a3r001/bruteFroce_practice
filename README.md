Configuración LOCAL (Recomendado)
1. Instala Node.js

Ve a https://nodejs.org
Descarga e instala la versión LTS
Verifica con: node --version en terminal

2. Crea la estructura de carpetas
proyecto/
├── server.js
└── public/
    └── index.html

3. Instala Express

Abre terminal en la carpeta proyecto
Ejecuta: npm install express

4. Inicia el servidor

En la misma terminal: node server.js
Deberías ver: Servidor ejecutándose en http://localhost:3000

6. Accede y prueba

Abre tu navegador: http://localhost:3000
Prueba con usuario: admin y contraseña: admin
